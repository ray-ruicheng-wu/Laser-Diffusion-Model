# 阶段工作报告模板

## 1. 阶段信息

- 阶段名称：
- 日期：
- 目标：
- 关联输出目录：

## 2. 本阶段实现内容

写明：

1. 新增了什么模块或脚本
2. 修改了哪些求解假设
3. 本阶段没有做什么

## 3. 生成结果与输出文件位置

至少列出：

1. 关键数值结果
2. 对应输出文件
3. 关键图表

## 4. 狗叫的审查记录

至少列出：

1. 审查时间
2. 审查对象
3. 检查了哪些公式、边界条件、守恒量和量纲
4. 发现了什么问题
5. 最终结论是 `通过 / 需修改后重审`

## 5. 与文书老哥提供的参考文献对比

至少列出：

1. 当前模型结果
2. 参考文献中的典型结果
3. 差距有多大
4. 这个差距更像来自哪一类物理缺失

## 6. 与现实/文献模型的误差分析

至少分析：

1. 热历史误差
2. 掺杂轮廓误差
3. source 剂量或边界条件误差
4. 模型阶次不足导致的误差

## 7. 每一步对应的物理公式

把本阶段真正采用的公式列出来，并标对应 `F-ID`：

1. 热方程
2. 光学热源
3. 相变
4. 扩散
5. 边界/界面条件

## 8. 当前风险

写明：

1. 还不确定的参数
2. 还没有并入的数学模型
3. 哪些结果不能过度解释

## 9. 下一步

给出：

1. 下一轮最值得做的事
2. 为什么
3. 对应可能新增的公式或文献

## 10. 研究线对标结论

这一节现在是强制项，至少要写：

1. 研究线是否判断本轮结果 `疑似有错 / 基本可信`
2. 研究线最值得优先修改的 1-3 项是什么
3. 每一项修改建议对应哪些文献
4. 每一项修改建议对应哪些 `F-ID` 或候选公式
5. 是否建议“先修正当前模型，再继续出新结果”
## 11. 片电阻分析标准产物

如果本阶段包含片电阻分析，阶段报告的“生成结果与输出文件位置”里必须额外列出这三个标准产物：

1. `silicon_p_profile_sheet_analysis.png`
2. `cumulative_p_dose_vs_depth.png`
3. `silicon_profile_analysis.csv`

同时要明确说明：`sheet dose` 统一从真实表面 `z=0` 开始积分。

如果模型里还存在初始未激活表面 `P` 层，也要在初始片电阻分析里单独记录它，并和 `active emitter` 分开写清楚。
