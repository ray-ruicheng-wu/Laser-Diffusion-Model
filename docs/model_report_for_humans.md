# Laser Doping Model Report For Humans

## 1. 这份报告是干什么的

这份报告不是给审稿人写的，也不是给代码作者自己看的。

它的目标只有一个：

把我们现在这个“激光诱导硅掺杂”模型，用普通人也能顺着读懂的方式讲清楚。

你可以把它理解成三部分：

1. 我们到底在模拟什么物理过程
2. 我们用了哪些公式，它们各自负责什么
3. 这些公式在代码里是怎么串起来工作的

这份报告覆盖的是当前项目已经真正落地的模型，不是未来可能会做的全部内容。

---

## 2. 我们到底在模拟什么

我们现在模拟的是下面这件事：

1. 硅片表面原本已经有一个基础的 `POCl3 diffusion` 发射极
2. 表面还覆盖了一层 `PSG`
3. 激光从上面打下来
4. 激光先把能量送进表面附近
5. 硅表面温度快速升高
6. 如果足够热，硅会部分熔化
7. 不管熔不熔，磷都会重新分布，只是速度差很多
8. 最后我们关心：
   - 温度多高
   - 有没有熔化
   - 磷浓度怎么分布
   - 结深有多深

一句话总结：

这是一个“先算热，再算相变，再算掺杂扩散”的耦合模型。

---

## 3. 为什么我们把模型分三层来建

我们没有一上来就做“全物理一次求完”的大模型，而是分成三层：

### 第一层：热模型

先回答：

- 激光把表面加热到多少
- 有没有到熔点
- 熔化持续多久
- 熔深有多深

如果连热都没算清楚，后面的掺杂结果一定不可信。

### 第二层：扩散模型

在已经知道温度历史以后，再回答：

- 磷怎么往硅里走
- 是主要停在表面，还是能往下走
- 最后结深有多大

### 第三层：表面层与工艺现实

这层是把实验现实补进去：

- `PSG` 不是“神秘表面层”，而是 `P2O5-SiO2` 玻璃
- 工业上激光前通常已经有一个基础 emitter
- `PSG/SiO2` 界面会抑制额外注入

这就是为什么我们后来又把：

1. `PSG/Si` 双层热模型
2. 非熔化固态扩散
3. 初始基础 emitter

逐步加了进来。

---

## 4. 当前模型的总流程

当前代码真正做的事情，可以按顺序理解成：

1. 给定激光参数
2. 把平均功率换算成单脉冲能量和 fluence
3. 计算热源在 `PSG` 和 `Si` 里的分布
4. 解热方程，得到 `T(z,t)`
5. 根据温度判断液相分数 `f_l(z,t)`
6. 根据温度和液相分数算扩散系数 `D(T, f_l)`
7. 根据表面 `PSG` source 和初始 emitter，解磷扩散方程
8. 从最终浓度曲线里读出结深

你也可以把它想成一条流水线：

`激光输入 -> 热 -> 相变 -> 扩散 -> 结深`

---

## 5. 第一部分公式：激光输入怎么进入模型

### 公式 F-017：平均功率换成单脉冲能量和 fluence

公式：

`E_pulse = P_avg / f`

`F = E_pulse / A_spot`

意思非常直接：

- 平均功率 `P_avg` 已知
- 重复频率 `f` 已知
- 那每个脉冲携带多少能量就知道了
- 再除以光斑面积，就得到每个脉冲的能量密度，也就是 `fluence`

它在模型里的作用是：

- 把实验室里常说的 `30 W`、`60 W`、`500 kHz`
- 变成热方程真正能用的输入

例如我们现在常用的工艺点：

- `60 W`
- `500 kHz`
- `95 um` 的 square flat-top 光斑

会得到：

- `120 uJ per pulse`
- `1.33 J/cm^2`

这一步看起来简单，但其实非常重要。

因为如果这一步错了，后面所有温度、熔深、掺杂结果都会一起错。

---

## 6. 第二部分公式：热是怎么计算的

### 公式 F-001：一维热传导方程

公式：

`rho * c_eff(T) * dT/dt = d/dz (k(T) * dT/dz) + Q(z,t)`

普通话解释：

- 左边是“这一小层材料温度升高需要多少能量”
- 右边第一项是“热会往上下传导”
- 右边第二项是“激光往这里塞了多少热”

这就是整个热模型的骨架。

它回答的问题是：

- 某个深度、某个时刻的温度是多少

### 为什么只做 1D

因为现在我们先关心深度方向上的事情：

- 能量在表面往里怎么走
- 熔深有多深
- 掺杂能走多深

横向光斑形状当然重要，但在第一版里我们先把横向信息压缩成一个面积，也就是只影响 fluence。

这就是“深度 1D 模型”的意义。

---

## 7. 第三部分公式：激光热源怎么分布

### 公式 F-002：Beer-Lambert 吸收

公式：

`Q(z,t) ~ A * I(t) * exp(-z / delta)`

意思是：

- 激光先打到表面
- 表面先吸一部分
- 越往深处走，能量越少
- 这个减少通常先用指数衰减表示

你可以把它想成“光进材料以后，一层层被吃掉”。

这里有两个关键量：

- `I(t)`：脉冲在时间上的形状
- `delta`：吸收深度

### 公式 F-016：分层热源

当前不再是简单的“裸硅”了，而是：

- 上面 `PSG`
- 下面 `Si`

所以热源分成两段：

`Q_psg ~ (1-R) I exp(-z/delta_psg)`

`Q_si ~ (1-R) I exp(-h_psg/delta_psg) T_int exp(-(z-h_psg)/delta_si)`

普通话解释：

1. 光先到最表面
2. 先被表面反射掉一部分
3. 进入 PSG 后又损失一点
4. 到 PSG/Si 界面再透过去一部分
5. 进入 Si 后继续被吸收

这一步的意义是：

把“表面是 `PSG` 不是裸硅”这件事实打进模型里。

---

## 8. 第四部分公式：相变怎么处理

### 公式 F-003：焓法 / 表观热容法

公式思想：

`cp_eff = cp + latent_heat_term`

普通话解释：

材料在熔化时，不是“温度一到熔点立刻跳过去”，而是会吃掉很多潜热。

如果显式追踪固液界面，模型会更复杂。

所以我们用了一个更稳妥的办法：

- 把潜热等效成一段很大的“额外比热”
- 这样数值上就能平滑地穿过熔点附近

### 公式 F-004：液相分数

思想是：

`f_l(T)` 在熔点附近从 `0` 平滑过渡到 `1`

意思：

- `f_l = 0` 表示完全固体
- `f_l = 1` 表示完全液体
- 中间值表示“接近熔化的过渡区”

这一步为什么重要？

因为后面的扩散系数要用它。

在我们的模型里，扩散强不强，不是只看“有没有熔深”，还看“局部到底有多少液相权重”。

这也是为什么后来你会看到：

- `max_melt_depth = 0`
- 但 `max_liquid_fraction` 还是可能大于 `0`

这不矛盾。

意思只是：

表面很接近熔化，甚至局部已经部分进入 mushy zone，但还没有形成我们定义下的“明确熔深”。

---

## 9. 第五部分公式：磷是怎么扩散的

### 公式 F-005：Fick 扩散方程

公式：

`∂C/∂t = ∂/∂z [ D(T, f_l) ∂C/∂z ]`

普通话解释：

- `C` 是磷浓度
- `D` 是扩散系数
- 如果某个地方浓度高、旁边浓度低，磷就会往低的地方跑

但这里的关键不是“会不会扩散”，而是：

`D` 到底有多大。

### 公式 F-006：固液混合扩散系数

公式：

`D_eff = D_solid * (1 - f_l) + D_liquid * f_l`

意思是：

- 完全固体时，用固态扩散
- 完全液体时，用液态扩散
- 中间状态按液相分数平滑混合

这一步非常关键。

因为激光掺杂之所以快，不是因为“磷突然会飞”，而是因为：

一旦局部进入液相，扩散系数会暴涨。

---

## 10. 第六部分公式：固态和液态扩散到底差多少

### 公式 F-019：固相 Arrhenius 扩散

公式：

`D_s = D0_s * exp(-Ea_s / (k_B T))`

普通话解释：

温度越高，固体里原子越容易动。

但在固体里，它还是动得很慢。

所以：

- 非融化条件下不是绝对零扩散
- 但通常很弱

这条公式是后来专门补进去的。

原因是你指出的那点非常对：

实验上经常不是“没熔化就完全零掺杂”。

所以模型里也不能这么粗暴。

### 公式 F-007：液相 Arrhenius 扩散

公式：

`D_l = D0_l * exp(-Ea_l / (k_B T))`

液相里为什么更快？

因为一旦熔化，硅原子结构松开了，磷在里面移动就会快很多。

这也是为什么：

- 轻微升温 -> 结深变化不大
- 一旦进入接近液相或部分液相 -> 掺杂会突然明显增强

---

## 11. 第七部分公式：表面 source 怎么进入硅

### 公式 F-008：有限库存 source

公式思想：

`Gamma_src(0) = C_src * h_src`

意思是：

我们没有把 `PSG` 做成一整个复杂玻璃网格去算磷怎么在里面跑，

而是先把它近似成一个“磷库存”。

这个库存一开始有多少，取决于：

- `PSG` 里的磷浓度
- 我们给定的等效 source 厚度

### 为什么这样做

因为对当前阶段来说，我们更关心的是：

- 它能往 Si 里送多少磷

而不是：

- 玻璃内部每一层的细节分布

这是一种“先把主效应抓住”的建模取舍。

---

## 12. 第八部分公式：界面注入怎么写

### 公式 F-009 / F-020：Robin 通量边界

公式：

`-D ∂C/∂z = h_m (C_eq,src - C_surf)`

普通话解释：

这条边界的意思是：

- 如果表面 source 这边“想给”的浓度更高
- 而硅表面当前浓度更低
- 那就会有一个净通量，把磷往硅里推

推动得多快，取决于：

- 当前扩散系数 `D`
- 界面传质长度 `L_tr`

通常我们把：

`h_m = D / L_tr`

### 为什么不用硬表面浓度

旧做法是：

`C(0,t) = C_source`

这等于假设：

- 玻璃一接触硅
- 表面立刻无限快达到 source 浓度

这通常太强了，不真实。

Robin 边界更像现实：

- 注入有能力上限
- 界面有阻力
- `SiO2` 夹层的抑制作用也能先折进这个边界里

### 现在这条边界怎么用

当前模型里：

- 固态时也用它
- 液态时也用它

差别不在“开不开”，而在：

- 固态时 `D` 很小，通量自然很弱
- 液态时 `D` 很大，通量自然会变强很多

这比“只有熔化才允许注入”更合理。

---

## 13. 第九部分公式：source 库存怎么更新

### 公式 F-010：总量守恒

公式：

`Gamma_src(t) = Gamma_src(0) - Integral C(z,t) dz`

普通话解释：

如果底部不漏，系统里总磷就应该守恒。

那就意味着：

- source 少了多少
- Si 里就多了多少

这一步是为了避免一个非常常见的数值错误：

- 硅里面的磷变多了
- 但 source 没有同步减少

或者反过来：

- 硅里面少了
- source 又没有回补

现在我们用总量守恒来更新库存，就是为了杜绝这种错账。

---

## 14. 第十部分公式：基础 emitter 怎么放进模型

### 公式 F-021：erfc 型初始轮廓

公式：

`C_init(z) = C_s * erfc(z / (2L))`

并且让：

`C_init(x_j) = C_background`

普通话解释：

工业上做过 `POCl3 diffusion` 以后，硅里本来就不是“零磷”。

它会有一个从表面往里逐渐下降的轮廓。

这种轮廓最常见、最经典的近似就是 `erfc` 形状。

你可以把它理解成：

- 表面最高
- 往里越来越低
- 到某个深度，降到背景掺杂
- 那个深度就是初始结深

我们现在就是用这个办法，把基础 emitter 放进模型里。

### 为什么这一步意义很大

因为如果你不把基础 emitter 放进去，

模型就会默认：

- 激光开始前，硅里一点磷都没有

这对工业工艺是不对的。

一旦把基础 emitter 加进去，很多事情会变得更接近现实：

1. 结深一开始就存在
2. PSG 额外注入不一定再主导“结深”
3. 它更可能主导“近表面浓度”

这和我们最近跑出来的结果是一致的。

---

## 15. 结深是怎么读出来的

当前定义很简单：

找到 `P` 浓度和 `Ga` 背景浓度相等的位置。

也就是：

`C_P(z_j) = C_Ga,background`

普通话解释：

- 如果某一层里 `P` 还是比 `Ga` 多
- 那它还属于 n 型这边
- 一旦降到和 `Ga` 一样
- 那就是结的位置

我们不是直接取某个网格点，而是做插值。

这样读出来的结深更平滑，也不容易被网格粗细卡住。

## 16. 当前模型到底是怎么建出来的

如果按工程路线看，我们的模型是这样长出来的：

### 第一步：裸硅单层热模型

先验证：

- 给激光以后，温度会不会合理升高
- 有没有假加热
- 有没有假熔化

这一阶段只做热，不做掺杂。

### 第二步：在温度历史上叠加扩散

这一步开始把：

- `PSG` source
- `P` 扩散
- `Ga` 背景

一起接进来。

### 第三步：修正基础物理错误

后来我们发现旧模型里有几类问题：

1. 库存更新会破坏守恒
2. 表面边界太强，像无限快注入
3. 固液界面扩散离散不稳
4. 不熔化被粗暴地当成零掺杂

所以逐步修成了现在这版。

### 第四步：加入 PSG/Si 双层热学

这一步的意义是：

不再把表面当成裸硅，而是明确有一层 `PSG`。

### 第五步：加入基础 emitter

这是最新一轮的重要升级。

因为它让模型终于开始接近工业流程：

- 先 `POCl3 diffusion`
- 再激光处理

## 17. 当前模型最重要的实验结论怎么理解

### 情况 A：没有基础 emitter，只靠 PSG source

如果热预算不够强，比如早期 `30 W` 情况，

模型会告诉我们：

- 即使允许固态扩散
- 注入也非常弱
- 很难形成明显结深

这说明：

单靠薄薄一层表面 source，在短时间里硬往里打，不一定够。

### 情况 B：提高到 60 W

到 `60 W` 以后，局部温度已经接近熔点。

虽然还没有明确熔深，

但局部液相分数已经不再是零。

所以你会看到：

- 结深开始明显出现
- 峰值浓度开始快速上升

这说明模型已经进入“接近熔化的过渡区”。

### 情况 C：加入基础 300 nm emitter

这一步最重要。

结果说明：

1. 结深马上就有了
2. PSG 再注入主要把表面浓度抬高
3. 但并没有明显把原来的 300 nm 继续推得更深

这支持一个非常重要的解释：

在工业现实里，基础 emitter 常常主导“结深”，

而激光和 PSG 更像在主导：

- 表面高浓度区
- 局部增强
- 选择性发射极的前表面电学改善

---

## 18. 当前模型还没做什么

这部分很重要，因为它决定了哪些结果能信到什么程度。

当前还没正式并入的东西包括：

1. 显式 `PSG/SiO2/Si` 三层光学
2. 熔池流动
3. 移动固液界面上的溶质守恒
4. `k(v)` 分配系数 / solute trapping
5. 更真实的工业初始 P 实测轮廓
6. 横向 2D 光斑和扫描路径

所以现在的模型更像：

一个已经物理自洽、能解释主要趋势的中间版本。

它已经比“拍脑袋调参数”强很多，

但还不是最终论文版。

---

## 19. 现在最该怎么理解界面抑制

你最近一直盯着一个很对的问题：

“基础 emitter 会不会改变界面抑制？”

当前模型和文献共同给出的答案是：

### 不会消失

`PSG/SiO2` 界面的阻挡作用不会因为下面已经有 emitter 就消失。

它仍然会限制：

- 额外从 PSG 注入多少磷

### 但重要性会变化

如果硅里已经有一个 300 nm 的基础结深，

那最终结深就不再完全靠“新注入多少磷”决定。

这时界面抑制更像是在决定：

- 表面额外还能不能再堆更多磷

而不是单独决定：

- 结深从 0 到 300 nm 能不能长出来

这正是我们最近算出来的现象。

---

## 20. 如果你想继续往下，最值当的是哪几步

如果目标是让模型更像工业现实，我建议优先级是：

1. 把基础 emitter 从 `erfc` 近似升级成更真实的工业轮廓
2. 把 `PSG/SiO2/Si` 界面显式建出来
3. 再决定是否要加 moving interface 和 `k(v)`

原因很简单：

- 现在影响结果最大的，已经不是“有没有扩散公式”
- 而是“初始轮廓对不对”和“界面怎么限制额外注入”

---

## 21. 一句话总结当前模型

我们现在的模型，不是一个“激光一打就从零长出结深”的玩具模型了。

它已经变成了一个更接近工业工艺的模型：

`基础 POCl3 emitter + PSG source + 532 nm 激光加热 + 部分相变 + 固/液扩散 + 界面受限注入`

当前最重要的结论是：

1. 没有基础 emitter 时，非融化条件下额外注入通常很弱
2. 提高功率会把系统推向接近熔化的过渡区
3. 一旦加入基础 emitter，结深主要由已有轮廓主导
4. PSG 和激光更多是在改表面高浓度区，而不一定显著继续推深结深

这就是我们现在这版模型真正“会说的话”。
